## 2. Procesos

> ### ▶️ Matar proceso
> ```shell script
> taskkill /F /IM <process-name>
> taskkill /F /PID <PID>
> ```
> Ejemplo: 
> - `taskkill /F /IM mysqld.exe`
> - `taskkill /F /PID 1234`
